================================================================================
QUICK TROUBLESHOOTING
================================================================================

Q: Animations not showing?
A: 1. Check jQuery loaded: Open console, type: typeof jQuery
   2. Check CSS classes match (product, item, btn, etc.)
   3. Refresh page with Ctrl+F5 to clear cache

Q: Animations running slow?
A: 1. Check browser performance (Chrome DevTools)
   2. Try closing other browser tabs
   3. Test on a different device
   4. Check internet connection

Q: Getting JavaScript errors?
A: 1. Open browser console (F12)
   2. Check error message
   3. Verify all 3 script lines added
   4. Verify script file paths are correct

Q: Animations conflicting with my CSS?
A: 1. Check for conflicting transitions in CSS
   2. Add !important to animation rules if needed
   3. Check for transform conflicts

Q: Mobile animations not working?
A: 1. Check mobile browser (Chrome, Safari)
   2. Test with DevTools mobile mode
   3. Verify touch events work
   4. Check performance on mobile
